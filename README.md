## About
***
这是慕课网上[vue+webpack打造todo应用](https://www.imooc.com/comment/935)课程的纯手打代码

# 使用方法

```git
git clone https://github.com/chenzhijie521143/vue-todo.git
```
## 使用除dist文件夹下的其他文件创建一个项目

##### 进入项目目录，运行
```
npm install
```
##### 然后执行
```
npm run dev
```
##### 开始开发项目
***
##### 最后，执行
```
npm run webpack
```
##### 即可打包出dist文件夹
